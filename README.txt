
VethaAI - Setup Instructions

1. To use offline:
   - Unzip the folder
   - Open index.html in your browser

2. To publish online (free using GitHub):
   - Create a free GitHub account at https://github.com
   - Create a new public repository (name: vethaai)
   - Upload all files
   - Go to Settings > Pages
   - Choose: Source = main, Folder = /root
   - Your chatbot will be live at: https://yourusername.github.io/vethaai

Questions welcome! Be blessed.
